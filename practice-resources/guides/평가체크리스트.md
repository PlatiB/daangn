# 📝 당근마켓 클론 코딩 실습 평가 체크리스트

## 🎯 개요
이 체크리스트는 **실습자의 자가진단**과 **강사의 평가**에 모두 사용할 수 있습니다.  
각 항목을 완료할 때마다 체크하여 진행상황을 확인하세요.

---

## 📋 1단계: 프로젝트 셋업 (15점)

### ✅ 기본 요구사항
- [ ] **React 프로젝트 생성 완료** (`create-react-app` 또는 `Vite` 사용)
- [ ] **프로젝트가 정상 실행됨** (`npm start` 성공)
- [ ] **필요한 패키지 설치** (styled-components, tailwindcss 등 중 선택)
- [ ] **Git 저장소 초기화** (선택사항이지만 권장)
- [ ] **폴더 구조 정리** (src/components, src/data 등)

### 📊 평가 기준
- **우수 (15점)**: 모든 요구사항 충족 + 추가 설정 (ESLint, Prettier 등)
- **보통 (10점)**: 기본 요구사항 모두 충족
- **미흡 (5점)**: 일부 요구사항 누락
- **불량 (0점)**: 프로젝트 실행 불가

---

## 🎯 2단계: HeroSection 컴포넌트 (25점)

### ✅ 기본 기능 구현
- [ ] **메인 타이틀 표시** "정자동에서 [동적텍스트] 찾고 계신가요?"
- [ ] **지역 표시** "정자동" 부분이 명확히 구분됨
- [ ] **검색창 구현** placeholder, 입력 기능
- [ ] **지역 선택 드롭다운** (최소한 UI만이라도)
- [ ] **서비스 선택 드롭다운** (중고거래, 알바, 부동산 등)

### ✅ 스타일링 요구사항
- [ ] **중앙 정렬** 제목이 화면 중앙에 위치
- [ ] **반응형 디자인** 모바일에서 줄바꿈 적용
- [ ] **검색창 스타일** 둥근 모서리, 적절한 패딩
- [ ] **일관된 폰트** 크기와 굵기 적용
- [ ] **색상 일치** 당근마켓 브랜드 컬러 활용

### ✅ 고급 기능 (보너스)
- [ ] **동적 텍스트 애니메이션** 키워드가 자동으로 변경됨
- [ ] **검색 기능** 실제 검색 동작 구현
- [ ] **드롭다운 상호작용** 선택 시 UI 변경
- [ ] **접근성 고려** aria-label, role 등 적용

### 📊 평가 기준
- **우수 (25점)**: 기본 + 스타일링 + 고급 기능 2개 이상
- **보통 (20점)**: 기본 + 스타일링 완료
- **미흡 (15점)**: 기본 기능만 구현
- **불량 (0점)**: 컴포넌트 미완성

---

## 📱 3단계: CategoryGrid 컴포넌트 (25점)

### ✅ 기본 구조
- [ ] **7개 카테고리 모두 표시** 중고거래, 알바, 부동산, 중고차, 동네업체, 동네생활, 모임
- [ ] **아이콘 + 텍스트 구조** 각 카테고리별 아이콘과 이름
- [ ] **그리드 레이아웃** 적절한 행/열 배치
- [ ] **클릭 가능한 링크** href 속성 또는 onClick 이벤트

### ✅ 반응형 구현
- [ ] **모바일 (1열)** 세로로 나열
- [ ] **태블릿 (2-3열)** 적절한 그리드
- [ ] **데스크톱 (4열)** 가로로 배치
- [ ] **여백과 간격** 일관된 spacing

### ✅ 시각적 품질
- [ ] **아이콘 통일성** 크기와 스타일 일관성
- [ ] **호버 효과** 마우스 오버 시 시각적 피드백
- [ ] **컬러 적용** 각 카테고리별 고유 색상
- [ ] **카드 스타일** 배경, 보더, 그림자 등

### 📊 평가 기준
- **우수 (25점)**: 모든 요구사항 + 뛰어난 시각적 완성도
- **보통 (20점)**: 기본 + 반응형 + 기본 스타일링
- **미흡 (15점)**: 7개 카테고리 표시 + 기본 레이아웃
- **불량 (0점)**: 카테고리 누락 또는 레이아웃 깨짐
