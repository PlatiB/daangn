# 🤖 AI 도구 활용 가이드 - 당근마켓 클론 실습

## 📋 시작하기 전에

이 가이드는 **당근마켓 클론 코딩 실습**에서 AI 도구(Cursor, Claude, ChatGPT 등)를 효과적으로 활용하는 방법을 안내합니다.

---

## 🚀 **1단계: 프로젝트 초기 설정**

### **AI에게 첫 질문하기**
```
당근마켓 메인 페이지를 React로 클론 코딩하는 실습을 진행하고 있습니다.

[프로젝트 요구사항]
- React 함수형 컴포넌트 사용
- 4개 주요 섹션 구현: HeroSection, CategoryGrid, PopularTags, PopularLocations
- 반응형 디자인 적용 (모바일/태블릿/데스크톱)
- 표준화된 데이터 구조 사용

React 프로젝트를 생성하고 기본 폴더 구조를 만들어주세요.
```

### **데이터 파일 생성 요청**
```
다음 데이터 구조로 src/data/mockData.js 파일을 생성해주세요:

1. categories: 7개 카테고리 (중고거래, 알바, 부동산, 중고차, 동네업체, 동네생활, 모임)
2. popularTags: 17개 인기 검색어
3. popularLocations: 20개 인기 지역
4. animatedKeywords: 동적 텍스트용 키워드
5. currentLocation: 현재 지역 정보

각 데이터에는 id, name, href, color 등의 속성을 포함해주세요.
```

## 🎯 **2단계: 컴포넌트별 구현**

```

- 메인 텍스트: "정자동에서 [동적키워드] 찾고 계신가요?"
- 동적 키워드: 3초마다 자동 변경
- 검색창과 드롭다운 포함

mockData.js의 데이터를 import해서 사용하고, useState와 useEffect로 애니메이션을 구현해주세요.

```
