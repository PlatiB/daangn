# 🥕 당근마켓 클론 코딩 실습 가이드

## 📋 실습 개요

**목표**: 당근마켓 메인 페이지를 React로 구현하여 프론트엔드 개발 역량 향상  
**소요시간**: 4-6시간  
**난이도**: 초급-중급  

## 🎯 학습 목표

- [x] HTML 구조 분석 및 컴포넌트 설계 능력
- [x] React 함수형 컴포넌트 작성
- [x] CSS-in-JS 또는 Tailwind CSS 활용
- [x] 반응형 웹 디자인 구현
- [x] Props를 통한 데이터 전달

## 📁 제공 리소스

### 1. HTML 구조 분석 자료
```
📂 .taskmaster/reference/rendered-html/
├── 📄 README.md              # 추출 가이드
├── 🎯 hero-section.html       # 메인 검색 영역
├── 🏷️ popular-tags.html       # 인기 검색어
├── 📱 category-grid.html      # 7개 카테고리
└── 📍 popular-locations.html  # 추천 지역
```

### 2. 구현해야 할 컴포넌트 목록
- `HeroSection` - 메인 타이틀 + 검색창
- `PopularTags` - 인기 검색어 태그들
- `CategoryGrid` - 7개 서비스 카테고리
- `PopularLocations` - 인기 지역 링크들

## 🚀 단계별 실습 진행

### **1단계: 프로젝트 셋업** ⏱️ 15분
```bash
# React 프로젝트 생성
npx create-react-app daangn-clone
cd daangn-clone

# 필요한 패키지 설치 (선택사항)
npm install styled-components
# 또는
npm install -D tailwindcss
```

**✅ 체크포인트**: `npm start`로 기본 React 앱이 실행되는가?

### **2단계: HeroSection 컴포넌트** ⏱️ 45분

#### 요구사항
- 지역 표시: "정자동에서"
- 동적 텍스트: "동네친구" (애니메이션 효과)
- 검색창: placeholder "검색어를 입력해주세요"
- 드롭다운: 지역선택 + 서비스선택

#### 구현 기준
```jsx
// 예시 구조
const HeroSection = () => {
  return (
    <section className="hero-section">
      <h1>정자동에서 <AnimatedText words={['동네친구', '맛집', '중고거래']} /> 찾고 계신가요?</h1>
      <SearchForm />
    </section>
  );
};
```

**✅ 체크포인트**: 
- [ ] 메인 타이틀이 화면 중앙에 표시되는가?
- [ ] 검색창이 올바르게 렌더링되는가?
- [ ] 반응형으로 작동하는가? (모바일에서 br 태그 적용)

### **3단계: CategoryGrid 컴포넌트** ⏱️ 60분

#### 요구사항
- 7개 카테고리: 중고거래, 알바, 부동산, 중고차, 동네업체, 동네생활, 모임
- 각 카테고리별 고유 아이콘 (SVG)
- 그리드 레이아웃 (모바일: 1열, 태블릿: 2열, 데스크톱: 3-4열)

#### 데이터 구조
```javascript
const categories = [
  {
    id: 'buy-sell',
    name: '중고거래',
    icon: '<svg>...</svg>',
    href: '/kr/buy-sell',
    color: '#FF6F0F'
  },
  // ... 나머지 6개
];
```

**✅ 체크포인트**:
- [ ] 7개 카테고리가 모두 표시되는가?
- [ ] 아이콘이 올바르게 렌더링되는가?
- [ ] 그리드 레이아웃이 반응형으로 작동하는가?
- [ ] 호버 효과가 적용되어 있는가?

### **4단계: PopularTags 컴포넌트** ⏱️ 30분

#### 요구사항
- 가로 스크롤 가능한 태그 리스트
- 태그별 링크 연결
- "인기 검색어" 라벨 (데스크톱에서만 표시)

#### 주요 기능
```jsx
const tags = [
  '에어컨', '에어컨청소', '노트북', '원룸', 
  '현대 중고차', '이사짐 알바', '근처 맛집'
  // ... 더 많은 태그들
];
```

**✅ 체크포인트**:
- [ ] 태그들이 한 줄로 나열되는가?
- [ ] 가로 스크롤이 작동하는가?
- [ ] 각 태그를 클릭할 수 있는가?

### **5단계: PopularLocations 컴포넌트** ⏱️ 30분

#### 요구사항
- 인기 지역 20개 표시
- Flex wrap 레이아웃
- 둥근 버튼 스타일

**✅ 체크포인트**:
- [ ] 지역들이 여러 줄로 배치되는가?
- [ ] 버튼 스타일이 일관되는가?

### **6단계: 메인 페이지 통합 및 최종 검토** ⏱️ 45분

#### 통합 작업
